# 🚀 CareerPilot AI

CareerPilot AI is an AI-powered placement preparation platform that helps students become placement-ready through resume analysis, interview preparation, personalized learning plans, and AI-driven career guidance.

---

# ✨ Features

- 📄 AI Resume Analyzer
- 🎯 Job Description Skill Gap Analysis
- 🤖 Personalized Interview Preparation
- 💻 Technical Interview Question Generation
- 🗣️ Behavioral Interview Question Generation
- 📚 AI-Generated Learning Roadmaps
- 📈 Placement Progress Tracking
- 🔐 JWT Authentication & Secure User Management

# 🛠️ Installation

## Clone the Repository

```bash
git clone https://github.com/Shubhang2k5/careerpilot-ai.git
cd careerpilot-ai
```

---

## Backend Setup

```bash
cd Backend
npm install
npm run dev
```

---

## Frontend Setup

```bash
cd Frontend
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the **Backend** folder:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
GEMINI_API_KEY=your_gemini_api_key
```

Create a `.env` file inside the **Frontend** folder:

```env
VITE_API_URL=http://localhost:5000
```
